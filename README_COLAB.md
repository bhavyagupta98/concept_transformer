# ConceptTransformer — Colab Quickstart

Quick instructions for running the repo in Google Colab (GPU runtime). This guide covers venv-free setup, downloading data separately, and running experiments.


Follow the cells in the file: Colab_Concept_Transformer.ipynb. This will run the setup, installation, training and test for both MNIST and CUB datasets.
Note: Due to google colab rendering and refresh issues the progress bar will be duplicated during training, however, everything is still working correctly.
